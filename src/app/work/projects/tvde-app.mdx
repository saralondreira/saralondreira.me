## Overview

A comprehensive Java-based management system developed for a TVDE (ride-hailing) company, built as an academic project for the Algorithms and Object-Oriented Programming course at ISEP. The application is designed to centralize and manage the core operations of a transportation business, handling the complex relationships between drivers, vehicles, clients, reservations, and trips.

[**View Code on GitHub ↗**](https://github.com/saralondreira/TVDE-Management-ISEP.git)

![TVDE App Main Menu](/images/projects/projects/tvde-app/menu.png)

## Key Features

- **Entity Management:** Full CRUD (Create, Read, Update, Delete) operations for Drivers, Vehicles, Clients, Reservations, and Trips with strict dependency constraints (e.g., preventing the deletion of vehicles that have associated trips).

![Creating a new TVDE entry](/images/projects/projects/tvde-app/newtvde.png)

- **Advanced Business Logic:** Functionality to seamlessly convert client reservations into active trips while strictly preventing schedule overlapping for vehicles and drivers.
- **Statistical Reporting:** Features to extract valuable business metrics, such as a driver's total earnings within a specific timeframe, average trip distances, and the most requested destinations.
- **Data Persistence:** Robust memory management using Arrays/ArrayLists, seamlessly integrated with File I/O operations to save and load application state, including support for managing multiple companies concurrently.

![Saving Data to File](/images/projects/projects/tvde-app/save.png)

## Technologies & Concepts Used

- **Programming Language:** Java
- **Core Concepts:** Object-Oriented Programming (OOP), Memory Management, Data Structures
- **Data Storage:** In-memory Collections (ArrayLists) & File I/O Persistence
- **Documentation:** Javadoc
